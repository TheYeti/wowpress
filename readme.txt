Add tooltips from \"WoWHead\" (http://www.wowhead.com/)

== Description ==
Adds Wowhead (http://www.wowhead.com) powered tooltips to your WordPress posts & pages using both shortcode and long links.

== Installation ==
1. Upload everything inside .zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
