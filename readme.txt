Add tooltips from "WoWHead" (http://www.wowhead.com/)

== Description ==

Adds Wowhead (http://www.wowhead.com) powered tooltips to your WordPress posts & pages

== Installation ==

* (1). Upload everything inside .zip to the `/wp-content/plugins/` directory
* 2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= Where is the data coming from? =
All data is loaded from http://*.wowhead.com using http://static.wowhead.com/widgets/power.js

= How to use? =
Since v1.0.0
Create a link to any item etc. on WoWHead ex. <a href="http://www.wowhead.com/item=89112">[Mote of Harmony]</a> and it will automatically transform into a tooltip.
